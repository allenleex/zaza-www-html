波场王炸GameFI（赛马游戏）：WIN NFT HROSE

  • 元宇宙NFT
  • 2021年12月02日05时

在区块链领域，GameFi 和 NFT 继续风靡全球。毫无疑问，不可替代代币（NFT）自诞生以来已经走过了漫长的道路。


随着世界上第一个“数字艺术市场”CryptoPunks 的推出，NFT 成为人们关注的焦点。当时，NFT 在区块链爱好者社区之外几乎没有受到关注。现在，NFT 很热。


然而，为什么会发生这种戏剧性的变化？


NFT游戏（GameFi）增长的主要原因之一是区块链行业提供了为玩家创造价值的能力。GameFi 是去中心化金融 (DeFi) 和博弈论的结合，它以玩赚钱的模式运作，而不是付费赢。这两种游戏模式的真正区别在于通过游戏玩法和游戏进程来分配经济奖励。


玩家可以通过投入时间参与此游戏来赚取收益。所有数据都存储在区块链中，因此，玩家拥有额外的好处，即他们个人拥有所有游戏资产，而不是游戏运营商或开发商。因此，即使服务器关闭，他们仍然拥有游戏中的所有资产。


综上所述，GameFi 游戏不仅可以娱乐人们，还可以让玩家在玩游戏的同时赚钱。

一、简介





WIN NFT HORSE是波场与APENFT、Winklink合作打造的“DeFi+NFT”赛马游戏。在很多游戏中，玩家仍然无法交易和兑现游戏内物品，从而导致巨大的损失。我们正在努力解决这个痛点。


与许多其他NFT游戏不同，WIN NFT HORSE 旨在让所有玩家从游戏中获得经济利益，而无需自己获得任何收益。


任何人都可以通过有趣的游戏享受 WIN NFT HORSE，并因其对生态系统的贡献而获得NFT奖励。玩家可以为他们的马匹繁殖、比赛、收集和买卖。


NFT是未来游戏行业的新载体，DeFi是未来游戏系统的新经济，为游戏生态提供可信赖的智能工具。


WIN NFT HORSE通过区块链技术与“NFT+DeFi”结合。最终，WIN NFT HORSE生态系统是通过可玩游戏和“NFT+DeFi”的结合而创建的。


这是一款非凡的 GameFi 产品，有可能超越 Axie Infinity。如果您错过了 Axie Infinity，请不要错过 WIN NFT HORSE。



NFT



区块链技术的发展使得 NFT 收藏者不再需要依赖受信任的第三方来维持其收藏的稀有性，而可以仅依赖于存储在区块链上的代码和文件的抗审查、不受信任、透明的智能合约。


一张 10M元的钞票可以兑换两张 5美元的钞票。一个金条可以换成另一个相同大小的金条。这些东西是可替代的。然而，NFT是独一无二的。


从本质上讲，NFT 是一种可收藏的数字资产，具有作为一种加密货币形式以及一种艺术或文化形式的价值。正如艺术被视为具有价值的投资一样，NFT 现在也是如此。


在WIN NFT HORSE中，每匹不同属性的马都是稀有的NFT，具有不同的经济价值。玩家可以通过盲盒和直接购买的方式拥有马匹NFT。



配种



在 WIN NFT HORSE 中，玩家可以购买、繁殖和收集称为“马”的可玩角色，这些角色具有不同的属性。


两匹马可以作为种马繁殖以产生幼马。当幼马成年后，它们的属性，包括战斗力和速度，将成倍增加，以释放它们的最大潜力。


由于每匹马的属性不同，其价值和能力也不同。如果玩家对他们的马匹的表现不满意，或者认为他们可以用他们独特的马匹换取大量代币，那么这个市场就是为你准备的。该市场允许玩家使用 TRX 买卖马匹。



战斗



WIN NFT HORSE 充满游戏乐趣。只有沉浸式的游戏体验，WIN NFT HORSE生态才能吸引足够多的玩家参与其中，从而为TRON区块链积累价值。


为满足不同玩家的需求，WIN NFT HORSE提供l多种不同类型的赛事，包括：挑战赛（PVE）、比赛（1v1 PVP）、竞赛（3v3 PVP）、邀请赛。





二、愿景





WIN NFT HORSE 是一种新型的 GameFi，它将奖励玩家在游戏和生态系统中投入的时间、精力和成本。WIN NFT HORSE 希望为其用户带来一个有趣、免费和开放的 GameFi 游戏平台。通过独特的算法模型，玩家可以在其中繁殖、管理、对抗和交易马匹。同时，玩家可以升级自己的马匹，变得更强大，创造更高的市场价值。


每匹赛马都有独特的基因，这决定了它的外观和特征。每匹马的基因数据都安全地存储在区块链网络的特定地址中，任何第三方都无法修改、复制或销毁。

WIN NFT HORSE 的所有数据都是在区块链上以开放、可信和去中心化的方式完成的。我们的目标是打造一个真正去中心化的 GameFi 游戏，这将是一个巨大的市场。我们希望通过赛马，连接世界各地的玩家，共同创造一个经济繁荣的世界。





三、优势





WIN NFT HORSE 是一款开放式赛马游戏。WIN NFT HORSE 开发团队将与社区合作，为相关游戏资产构建丰富的功能。


WIN NFT HORSE 具有以下优势：


拥有丰富的博弈机制，完全公平公开的收款市场、交易市场和投资市场。


本游戏具有公开透明的运行环境，以公平公正为核心价值观，让所有玩家随时回顾游戏。玩家拥有相同的地位和权力，这不会造成玩家之间的不平等。众所周知，在现有的一些游戏中，其算法并不公开、不透明，甚至不公平。如果玩家处于不平等的游戏环境中，最终会造成用户的流失。


用户粘性高，交易市场功能不断完善，将有助于推动整个WIN NFT HORSE的生态繁荣。鼓励玩家参与游戏并获得奖励，并向新玩家推荐游戏。


科学稳定的经济体系可以增加玩家的积极性。每一个进入WIN NFT HORSE生态的玩家，不仅会被这款赛马游戏所吸引，在深入游戏的过程中还能获得意想不到的奖励。





四、玩法





挑战 (PVE) 是一种人机对战的游戏玩法，类似于 RPG 中的推、破和爬塔。在PVE模式下，玩家的马匹将根据他们成功通过的等级获得经验。经验可用于升级马匹，提升四维属性。



挑战



随着经验值的增加，玩家的马匹会升级，从而增强PVE模式下的整体实力，从而更容易通过更高的等级，并获得更多的NFT作为奖励。通过奖励优胜者丰富的 NFT，有效鼓励所有玩家完成 PVE 日常任务，积极参与 PVE 游戏。


需要提醒的是，只有当玩家赢得当前关卡才能开启下一关卡。


然后，玩家应根据该级别的地形和天气条件选择合适的马匹进行比赛。


选好马匹后，根据马匹和场地特点选择赛马策略。




比赛


比洞赛（1v1 PVP）是玩家选择一匹马和策略，然后互相对战，单场比赛最多15匹马互相对战。最终获胜者将获得 NFT 代币作为奖励；失败者不会扣除任何代币或积分。


PVP 随机匹配模式鼓励玩家通过慷慨的 NFT 代币作为奖励进行竞争，以期提高玩家对游戏内收入的期望。


PVP模式相对于其他几类游戏来说是比较直接的，PVP是展现整个WIN NFT HORSE的可玩性，以及给予玩家优越感和满足感的主要方式。



竞技（3v3 PVP）



与单匹马的比洞赛（1v1 PVP）不同，竞技赛（3v3 PVP）需要设置您自己的马匹阵容。玩家需要从自己的马匹中选择3匹马与其他玩家的阵容进行离线PVP。（注：如果玩家的马匹少于3匹，将无法与其他玩家对战）


竞技Play以胜负决定积分，战斗机制为KOF模式。



关于马匹出现的顺序，玩家需要选择具体的马匹出现顺序并保存设置，因为下次进入马匹选择界面会保留上次的选择。


如果玩家不设置出场顺序，游戏中默认出场顺序为仓库中战力最高的3匹马，从高到低排列。


此外，如果已保存设置的马匹已售出，则将使用剩余战斗力最强的马匹来填补位置。


WIN NFT HORSE将为每个赛季积分排名前列的玩家发放丰厚的WIN代币奖励。这是赛马系统产生价值的重要渠道。



邀请赛



邀请赛根据报名费的高低分为三个级别。不同级别的锦标赛会使用不同的锦标赛场景，越高级的锦标赛场景越华丽。


1. 普通环需要1000NFT及以下。

2. 大师对决需要1001~5000NFT。

3.大决斗报名需要5001NFT及以上。



比赛级别根据报名费进行评判，不同级别使用不同场地。


1. 普通环使用1000M路线的普通场地。

2. 大师对决使用2200M球场的中级场地。

3.使用3000M球场的高级场地进行大决斗。


一次邀请赛参赛人数最多为15人，最少为2人。所有比赛将在报名开始后一小时内进行，在此期间可以邀请所有选手或直接打开邀请赛名单参加比赛。


注册将在启动后 1 小时内完成。报名结束前，主办方必须选择开始或取消比赛。如果没有选择，戒指将被取消，赞助商和所有参与者将通过邮件退还报名费。





五、马养殖





马匹的饲养是整个WIN NFT HORSE游戏最重要的环节之一，由于其随机性高，不同马匹之间的巨大价格差异，增加了“运气”在游戏中的重要性。


当玩家培育一匹马时，每个基因都有机会传递给后代。当然，不同种族的基因结合也会产生基因突变，这是随机的。


此外，每匹马都有可变性。我们尊重生物学的自然规律，使品种符合遗传的生物学规律，具有随机性和可塑性。我们赋予每匹马升级的能力。玩家可以使用积分升级马匹，让他们在战斗中更加强大。



种族（血统）



在搞清楚马匹的繁育之前，一定要清楚地了解什么样的马匹在游戏中更有价值。等级越高，价值越高。



马的种族（血统）将是马育种的关键参数。马的属性和技能由系统决定，有一定的随机属性。


配种时，如果两匹马的血统相同，后代也会100%的血统相同。

一 例如，如果马A是伯伯，马B是伯伯，那么马C就是伯伯。

配种时，如果两匹马的血统不同，25%的后代会继承高级马的血统，75%的后代会继承低级马的血统。


二例如，如果马 A - 阿拉伯马和马 B - 纯种马，那么 25% 的育种结果是纯种马，75% 的结果是阿拉伯马。



性别



繁殖结果：50%几率为雄性，50%几率为雌性。



颜色



幼马会随机继承公马或母马的原色，不可能继承公马或母马本身没有的任何颜色。



部分


马的部位分为底纹、左前腿、左后腿、右前腿、右后腿、头纹。

幼马会随机继承公马或母马的部分，并且不可能继承公马或母马本身没有的任何特征的部分。



养殖成本



每匹马都可以与另一匹马交配。为了避免人口过多，每匹马最多可以繁殖七次。每次育种之间需要一定的时间间隔。


此外，育种需要一定数量的NFT和WIN代币。消耗的代币数量取决于马匹的繁殖次数。



繁殖周期


一匹小马变成一匹种马需要多长时间？


成为一匹种马需要5天时间。


5 天后，您的小马将长成一匹种马。这些马可用于繁殖、战斗或出售。





六、稳定





就像现实世界中的宠物一样，马可以通过繁殖来繁殖后代。



马厩的功能是提供一个平台，让玩家可以了解自己马匹的属性、场地适应性等数据，以及自己拥有的马匹的概况。


玩家可以在“稳定”界面看到马匹的具体属性，如名称、等级、战斗力、场地适应性、距离适应性、速度、耐力、爆发和平衡属性、技能等。


马的等级和四维属性只在PVE中有效，而马在比洞赛中只显示1级和初始属性。马匹的适合度仅根据比赛信息（不可见）影响马匹在比赛时的4维属性，距离适合度只显示最适合的距离，即推荐参加的比赛长度。
