热讯：Blockchain.Com 推出NFT市场

  • BiBiNFT
  • 2021年12月05日05时



最近，Blockchain.com 加入了大量交易所，这些交易所推出了不可替代代币 (NFT) 市场。12 月 2 日，该公司宣布将推出Blockchain.com NFT 市场，部分用户2号可以注册 Beta 版访问权限。





Blockchain.com NFT Marketplace 推出 Beta Access



不可替代的代币 (NFT) 资产已成为一个价值 10 亿美元的行业，似乎这项技术将成为未来几年的支柱。随着 NFT 交易在 2021 年呈指数级增长，今年许多交易所推出了 NFT 市场。Crypto.com、Coinbase 和 FTX 等知名交易所已经在 Opensea 和 Rarible 等巨大的独立 NFT 市场中展示了 NFT 市场。现在，金融服务公司Blockchain.com 宣布了其 NFT 市场。


“我们想让进入 NFT 市场就像进入区块链市场一样容易，”Blockchain.com 的公告细节。“借助Blockchain.com NFT 市场（测试版），你无需离开Blockchain.com 就可以浏览、购买、出售和安全存储 NFT。”




公司发布 NFT Explorer



Blockchain.com 的 NFT 市场发布公告是在该公司最近收购阿根廷投资平台 Sesocio 之后发布的。此次收购使Blockchain.com 的全球员工人数达到 400 名，并计划在拉丁美洲建立实体店。从现在开始，人们可以注册该公司新的 NFT 市场的 Beta 版访问权限，该市场将允许用户在他们的Blockchain.com 非托管钱包中“购买、出售和存储 NFT”。


该公司还推出了 NFT 浏览器，可提供有关不可替代代币的信息，例如当前价格、底价、当前 NFT 所有者的信息和最后销售价格。Blockchain.com 的 NFT 浏览器信息源于托管在 Opensea NFT 市场上的基于以太坊的 NFT。公告博客文章还为读者提供了一些见解，向个人展示了如何在Blockchain.com 市场上购买 NFT 的分步指导。
